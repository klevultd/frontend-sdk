# Klevu Apps

Klevu App components are full solutions. Just add one component to your code and you have full working Klevu discovery tool done.

Basic philosophy with these components is that you can modify them just a little bit with attributes. But if you wish to do
anything more complicated you need to use **Klevu Components**
