## Slider with tooltips

By setting the `formtTooltip` property, you can format the tooltip values.

<Canvas>
  <Story
    name="Slider with tooltips"
    args={{
      min: 0,
      max: 100,
      start: 10,
      end: 90,
      showTooltips: true,
    }}
  >
    {(args) => html`<klevu-slider
      end=${ifDefined(args.end)}
      max=${ifDefined(args.max)}
      min=${ifDefined(args.min)}
      show-tooltips=${ifDefined(args.showTooltips)}
      start=${ifDefined(args.start)}
      .formatTooltip=${(value) => `$${value.toFixed(0)}`}
    ></klevu-slider>`}
  </Story>
</Canvas>
