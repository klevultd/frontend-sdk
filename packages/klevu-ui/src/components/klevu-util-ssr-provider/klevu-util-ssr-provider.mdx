# Klevu-util-ssr-provider

<Markdown>{Stories.description}</Markdown>

SSR Provider example will transform components data to a string that set to attribute of the this utility component.

When a client side reads this provider and finds the attribute, it will transform the string to a JSON object and pass it to the components.
