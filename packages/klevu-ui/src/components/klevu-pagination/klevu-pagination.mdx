# Klevu-pagination

<Description>{meta.description}</Description>

Pagination can be build simply with just providing the min, max and current values.

<Canvas>
  <Story
    name="Pagination"
    args={{
      min: 1,
      max: 6,
      current: 3,
    }}
  >
    {(args) =>
      html`<klevu-pagination
        current=${ifDefined(args.current)}
        max=${ifDefined(args.max)}
        min=${ifDefined(args.min)}
        .queryResult=${args.queryResult}
      ></klevu-pagination>`
    }
  </Story>
</Canvas>

## Pagination from query results

Pagination can be automatically build from the @klevu/core query results. Just provide the query results property to the component.

<Canvas>
  <Story
    name="Pagination from results"
    args={{
      queryResult: fullMockRequest.queryResults?.[0],
    }}
  >
    {(args) => html`<klevu-pagination .queryResult=${args.queryResult}></klevu-pagination>`}
  </Story>
</Canvas>
