## Pagination from query results

Pagination can be automatically built from the @klevu/core query results. Just provide the query results property to the component.

<Canvas of={Stories.PaginationFromResults} />

### Here is the code for the above example in React (@klevu/ui-react):

#### searchResponse - the @klevu/core query results object

```jsx
import React from "react"
import { KlevuPagination } from "@klevu/ui-react"

const MyComp = () => {
  return <KlevuPagination queryResult={searchResponse.queryResults?.[0]}></KlevuPagination>
}
```
